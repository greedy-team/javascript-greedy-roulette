## CSS 스타일 및 구조 추가

- 스타일링은 자유롭게 구성했습니다.
- 제공된 HTML 구조와 id/class는 변경하지 않았습니다.
- 비활성화된 버튼은 시각적으로 구분되도록 디자인 했습니다.
- 베팅 성공/실패 결과는 시각적으로 구분되도록 디자인 했습니다.
- 결과 색상은 시각적으로 구분되도록 디자인 했습니다.

---

#### 초기화

- 현재 자금에 `10,000`을 표시 했습니다.
- 현재 라운드는 `0`을 표시 했습니다.
- 다시 시작 버튼, 결과 영역은 숨김 처리 했습니다.
- 베팅 버튼과 중단 버튼은 활성화 되게 했습니다.

---

### 사용자 입력 구현

- 베팅 금액 입력: 1원 이상의 정수를 입력한다.
- 베팅 실행 버튼 기능을 구현했습니다.
- 게임 중단 버튼 기능을 구현했습니다.
- 다시 시작 버튼 기능을 구현했습니다.

---

#### 베팅 진행

- 현재 자금은 베팅 금액만큼 즉시 차감되어 표시했습니다.
- 베팅 버튼과 중단 버튼은 비활성화 했습니다.
- 결과 영역과 아래 문구를 표시했습니다.

```markdown
룰렛을 돌리는 중...
```

#### 베팅 결과 메시지 표시

- 성공, 실패 여부에 따라 메시지를 표기했습니다.

```markdown
베팅 성공! +{획득 금액}원
```
```markdown
베팅 실패! -{손실 금액}원
```

---

#### 베팅 결과 화면 표시(2초 후)

- 현재 라운드가 1 증가하여 표시 했습니다.
- 결과 영역에 아래 정보를 표시했습니다.

```markdown
룰렛 결과: {색상}
```

- 베팅 버튼과 중단 버튼을 다시 활성화 했습니다.

---

## 예외 처리

- 유효하지 않은 입력이 들어오면 `alert`로 에러 메시지를 표시했습니다.
- 에러 발생 시 다음의 상태는 변경되지 않도록 주의 했습니다: 현재 자금, 현재 라운드, 버튼 활성/비활성 상태

---

#### 게임 종료

- 조건 충족 시 게임이 종료되도록 했습니다.
- 종료 시 결과 영역에 최종 결과가 표시 했습니다.

```markdown
게임 종료
최종 자금: {금액}원
플레이한 라운드: {라운드 수}
```

- 베팅 입력 영역(색상 선택, 금액 입력, 버튼)은 숨김 처리했습니다.
- 다시 시작 버튼이 보이도록 했습니다.
