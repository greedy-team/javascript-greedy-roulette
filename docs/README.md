# 탐욕의 룰렛 게임 기능 목록

1. 게임 초기화
   - 초기 자금 10,000원 설정
   - 라운드 0 표시
   - 결과 영역 숨김
   - 다시 시작 버튼 숨김
   - 베팅 버튼과 중단 버튼 활성화 상태
2. 룰렛 구성
   - 40칸: YELLOW(21), GREEN(10), BLUE(6), PURPLE(2), RED(1)
   - 각 칸 동일 확률(2.5%)로 선택
   - 배당: YELLOW x1, GREEN x3, BLUE x5, PURPLE x10, RED x20
3. 베팅 입력 검증 → 실패 시 alert으로 에러 메시지를 보여주고, 다시 입력할 수 있게 한다
   - 색상이 선택되지 않은 경우
   - 베팅 금액이 1원 이상의 정수가 아닌 경우 - 양수 포메팅
   - 베팅 금액이 현재 자금을 초과하는 경우 - 현재금액과 비교
   - 에러 발생 시 자금, 라운드, 버튼 상태 변경 없음 - 검증 후 버튼 로직 실행 (4번 만들 때 같이 진행)
4. 베팅 실행 (버튼 클릭 직후)
   - 자금에서 베팅 금액 즉시 차감하여 표시
   - 베팅 버튼과 중단 버튼 비활성화
   - 결과 영역에 "룰렛을 돌리는 중..." 표시
5. 룰렛 결과 출력 (2초 후)
   - 라운드 1 증가하여 표시
   - 결과 영역에 "룰렛 결과: {색상}" 표시
   - 결과 색상 시각적으로 구분
   - 베팅 버튼과 중단 버튼 재활성화
6. 베팅 성공/실패 처리
   - 성공: 베팅 금액 + (베팅 금액 × 배당) 획득, "베팅 성공! +{획득 금액}원" 표시
   - 실패: "베팅 실패! -{손실 금액}원" 표시
7. 게임 종료
   - 중단 버튼 클릭 시 게임 종료
   - 자금 0원 이하(파산) 시 "게임이 곧 종료됩니다." 표시 후 2초 뒤 종료 화면 전환
   - 종료 화면: "게임 종료 / 최종 자금: {금액}원 / 플레이한 라운드: {라운드 수}"
   - 베팅 입력 영역 숨김
   - 다시 시작 버튼 표시
8. 다시 시작
   - 다시 시작 버튼 클릭 시 초기 상태로 리셋

## 코딩 컨벤션

- var 사용 금지, const와 let 사용
- 전역 변수 사용 금지
- 외부 라이브러리 사용 금지, 순수 Vanilla JS만 사용
- Airbnb 자바스크립트 코드 컨벤션 준수
- indent depth 2 이하
- 함수(메소드)는 한 가지 일만 하도록 최대한 작게
- 함수(메소드) 길이 15라인 이하
- import 문으로 모듈화
