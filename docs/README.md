# 탐욕의 룰렛 게임 기능 구현 미션

## 기능 구현 목록 


기능
1. 게임 초기화 (init)
    - 초기 자금 10,000원, 라운드 0으로 설정.
    
    - 베팅 영역(색상 선택, 금액 입력, 버튼) 표시.
    
    - 결과 박스 및 다시 시작 버튼 숨김 처리.

2. 베팅 입력 및 유효성 검사
    - 색상 선택 확인: 선택되지 않았을 경우 alert 표시 후 중단.
    
    - 금액 입력 확인: 숫자가 아니거나 0 이하일 경우 alert 표시 후 중단.
    
    - 잔액 비교: 보유 금액보다 베팅 금액이 크면 alert 표시 후 중단.
    
    - 유효성 검사 통과 시 즉시 현재 자금에서 베팅 금액 차감.

3. 베팅 진행 상태 표시 (즉시 실행)
    - 베팅하기/중단 버튼 숨김 처리 (중복 클릭 방지).
    
    - 결과 영역 노출 및 "룰렛을 돌리는 중..." 메시지 출력.

4. 룰렛 결과 도출 (2초 후 실행)
    - 라운드 수 증가: 결과 발표와 동시에 현재 라운드 +1 반영.
    
    - 확률 기반 색상 결정: 1~40 사이의 난수를 생성하여 칸 수에 따른 확률 적용.

5. 결과 판정 및 상금 지급
    - 성공 판정: 선택한 색상과 결과 색상이 일치할 경우 상금 계산 및 지급.

        상금 공식: 베팅 금액 + (베팅 금액 × 배당) (원금 회수 포함)

    - 실패 판정: 일치하지 않을 경우 손실 금액 메시지 출력.

    - UI 반영: 결과 색상을 소문자 변환하여 CSS 색상으로 시각적 구분 출력.

6. 게임 종료 로직 (endGame)
    - 파산 체크: 결과 반영 후 자금이 0원 이하일 경우 안내 메시지 출력 후 2초 뒤 종료.

    - 사용자 중단: 중단 버튼 클릭 시 2초 뒤 종료 화면 전환.

    - 종료 화면 전환: 베팅 입력창을 숨기고 최종 자금 및 누적 라운드 수 표시.

7. 재시작 기능
    - '다시 시작하기' 버튼 클릭 시 init 함수를 호출하여 모든 상태 초기화.

    - 비활성화되었던 버튼들의 상태(disabled) 활성화.
