# 기능 목록

## 게임 초기화

- [ ] 플레이어 초기 자금 10,000원 설정
- [ ] 라운드 수 초기화 (0)

## 게임 진행

- [ ] 매 라운드마다 베팅 진행 또는 게임 중단 선택
- [ ] 베팅 시 색상 입력
- [ ] 베팅 시 금액 입력
- [ ] 베팅 금액 자금에서 차감

## 룰렛 규칙

- [ ] 룰렛 20칸 구성
  - YELLOW: 10칸 (50%, x1 배당)
  - GREEN: 5칸 (25%, x3 배당)
  - BLUE: 3칸 (15%, x5 배당)
  - PURPLE: 1칸 (5%, x10 배당)
  - RED: 1칸 (5%, x20 배당)
- [ ] 룰렛 결과 2초(2000ms) 후 출력

## 베팅 결과 처리

- [ ] 베팅 성공: `베팅 금액 + (베팅 금액 × 배당)` 획득
- [ ] 베팅 실패: 베팅 금액 잃음

## 게임 종료

- [ ] 자금 0원 시 게임 종료
- [ ] 플레이어가 게임 중단 선택 시 종료
- [ ] 최종 자금 출력
- [ ] 플레이한 라운드 수 출력

## 입력 검증

- [ ] 잘못된 색상 입력 시 alert 에러 메시지 표시 후 재입력
- [ ] 잘못된 베팅 금액 입력 시 alert 에러 메시지 표시 후 재입력
  - 보유 자금보다 큰 금액
  - 0 이하의 금액
  - 현재 자금보다 큰 값
  - 자연수가 아닌 값
